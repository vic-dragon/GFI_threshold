Readme

Densities.R: Densities for Metropolis-hasting algorithm.
Fiducial.R: Some functions to calculate fiducial density
GenerateSamples.R: For numerical experiments, sampling scheme
ModelSelection.R: Fiducial density for model selection.
pvaluemodel.R: Competing method 1.
Simulation.R: To implement numerical experiments with two competing methods and our method.

n6m5.R, n10m10.R, n10m20.R, n20m50.R, n50m100.R: To run simulation for each different scenario.

Resultsummary.R: With outputs(Rdata) from the above R code, this script helps to organize the result.
